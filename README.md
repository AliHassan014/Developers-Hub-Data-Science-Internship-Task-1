# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
The objective of this task is to understand how to load, inspect, summarize, and visualize a dataset using Python.

## Dataset
The Iris dataset was used for this task. It contains measurements of iris flowers, including:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

## Approach
The following steps were performed:

1. Loaded the dataset using the pandas library.
2. Examined the dataset structure using:
   - `shape`
   - `columns`
   - `head()`
3. Checked the dataset for missing values.
4. Performed Exploratory Data Analysis (EDA).
5. Created visualizations using matplotlib and seaborn:
   - Scatter Plot
   - Histogram
   - Box Plot

## Results and Insights
- The dataset was successfully loaded and explored.
- Scatter plots helped visualize relationships between variables.
- Histograms showed the distribution of numerical features.
- Box plots highlighted the spread of data and potential outliers.
- The task provided practical experience with pandas, matplotlib, and seaborn.
